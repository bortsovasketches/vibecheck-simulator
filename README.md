**Vibecheck Simulator** is an advanced content resonance analysis tool powered by simulated AI personas. It allows you to test your content against a diverse panel of AI critics, skeptics, and fans before you publish.

![Vibecheck Simulator](/src/assets/avatars/avatar1.png)

## Features

-   **AI Persona Simulation**: Generate realistic audience personas with unique backgrounds, goals, and pain points.
-   **Sci-Fi Futurist Aesthetic**: Experience a "Midnight Electric" interface with futuristic persona names (e.g., XAe-12, Nova Prime).
-   **Dynamic Avatars**: Personas are assigned unique, abstract avatars for a more immersive experience.
-   **Deep Analysis**: Get detailed feedback on resonance, confusion points, and tone analysis (Defense, Jargon, Empathy, Clarity).
-   **Scenario Modes**:
    -   **Standard Mode**: For general audience testing.
    -   **Crisis Mode**: Simulates valid critics and PR nightmares to help you stress-test sensitive content.
-   **Report Generation**: Synthesizes all feedback into a comprehensive "Vibe Report" with a Go/No-Go recommendation.

## Tech Stack

-   **Frontend**: React, Vite, TypeScript
-   **Styling**: Tailwind CSS, Shadcn UI, Framer Motion
-   **State Management**: Zustand
-   **AI Integration**: Google Gemini (via Vercel AI SDK)
-   **Desktop App**: Electron

## getting started

1.  Clone the repository:
    ```bash
    git clone https://github.com/bortsovasketches/vibecheck-simulator.git
    cd vibecheck-simulator
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  To build for Electron:
    ```bash
    npm run electron:build
    ```
